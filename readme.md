# wat

marketing tool for twitter

# how

```bash
git clone https://github.com/syzer/twitter-marketing-bot.git
cd twitter-marketing-bot
npm install
cp .env.dist .env
# get your key from https://dev.twitter.com/resources/signup
# edit .env with all the kys 
npm start
```
